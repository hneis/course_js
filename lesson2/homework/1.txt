1. Дан код:
1) var a = 1, b = 1, c, d;
2) c = ++a; alert(c);           // 2
3) d = b++; alert(d);           // 1
4) c = (2+ ++a); alert(c);      // 5
5) d = (2+ b++); alert(d);      // 4
alert(a);                    // 3
alert(b);                    // 3
Почему код даёт именно такие результаты?
1. обычное присвоение
2. сначал инкриментируем a(2) затем присваеваем ее в с;
3. Сначала присваеваем 'c' значение которое хранится в 'b', затем инкриментируем 'b');
4. инкриментируем 'a' и прибавляем к 2, после чего присваеваем получившееся значение 'c'
5. 2+b присваиваем 'c', после чегео инкриментируем b
